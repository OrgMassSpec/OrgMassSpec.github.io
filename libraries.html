Mass Spectral Libraries

The mass spectral libraries below currently focus on electron impact (EI) spectra of anthropogenic environmental contaminants and halogenated natural products. The libraries document and organize the evidence for compound identifications, and may also contain interesting but unidentified mass spectra. Each library is provided as a PDF report containing the annotated spectra, a single text file containing all spectra in NIST MSP text format, and an R package. The PDF report allows the spectra to be viewed without specialized software. The MSP file can be used to import spectra into other software. The R package stores the mass spectra and associated metadata, and generates the PDF report and text file.

California Dolphin Blubber Contaminant Library

The original PDF version of this library appears as supporting information for the manuscript referenced below.

Report: SpecLibDolphin2014.pdf
NIST MSP: SpecLibDolphin2014.zip
R Package: SpecLibDolphin2014 Source Package
Reference: "Nontargeted analysis reveals hundreds of unmonitored bioaccumulative compounds in bottlenose dolphins from Southern California" (submitted)
Authors: Nellie J. Shaul, Nathan G. Dodder, Lihini I. Aluwihare, Susan A. Mackintosh, Keith A. Maruya, Susan J. Chivers, Kerri Danil, David W. Weller, Eunha Hoh
Samples: Blubber from two ecotypes of common bottlenose dolphins (Tursiops truncatus) fatally stranded in the Southern California Bight from 1995 to 2010.
Instrument: GCxGC-TOF in EI mode.

San Diego Bay Bird Egg Contaminant Library

The original PDF version of this library appears as supporting information for the manuscript referenced below.

Report: Upcoming
NIST MSP: Upcoming
R Package: Upcoming Source Package
Reference: "Identifying Bioaccumulative Halogenated Organic Compounds Using a Nontargeted Analytical Approach: Seabirds as Sentinels"
Authors: Christopher Millow, Susan A. Mackintosh, Rebecca L. Lewison, Nathan G. Dodder, and Eunha Hoh

Massachusetts Dolphin Blubber Contaminant Library

The original PDF version of this library appears as supporting information for the manuscript referenced below. Note the PDF version here has different page numbers.

Report: SpecLibDolphin2011.pdf
NIST MSP: SpecLibDolphin2011.txt
R Package: SpecLibDolphin2011 Source Package
Reference: "Nontargeted Comprehensive Two-Dimensional Gas Chromatography/Time-of-Flight Mass Spectrometry Method and Software for Inventorying Persistent and Bioaccumulative Contaminants in Marine Environments", Environmental Science and Technology, 2012, 46 (15), 8001-8008. DOI: 10.1021/es301139q.
Authors: Eunha Hoh, Nathan G. Dodder, Steven J. Lehotay, Kristin C. Pangallo, Christopher M. Reddy, and Keith A. Maruya
Sample: Blubber from a common dolphin (Delphinus delphis) fatally stranded January 2006 in Orleans, Massachusetts.
Instrument: GCxGC-TOF in EI mode.

California Sea Lion Contaminant Library (Example)

This library was originally provided as proof-of-concept.

Report: SpecLibExample.pdf
NIST MSP: SpecLibExample.txt
R Package: SpecLibExample Source Package
Authors: Eunha Hoh and Nathan Dodder
Sample: Blubber from a male sea lion (Zalophus californianus) fatally stranded in the year 2000 in southern California.
Instrument: GCxGC-TOF in EI mode. A description of the analytical method is in the package help.
