Mass Spectral Libraries

Mass spectral libraries are provided as companion packages to OrgMassSpecR. The library content is currently focused on electron impact (EI) spectra of anthropogenic environmental contaminants and halogenated natural products. The libraries are intended to document and organize the evidence for compound identifications, and thus have a somewhat different focus than standard reference databases such as the NIST/EPA/NIH library. For example, the OrgMassSpecR libraries store fragment ion identifications obtained by the manual interpretation of mass spectra and allow organization based on compound class and other parameters. The libraries may also contain interesting but unidentified mass spectra.

Each library package generates a PDF report containing the annotated spectra, and a single text file containing all spectra in NIST MSP format. The PDF report allows the spectra and identification evidence to be viewed without specialized software. The MSP file can be used to import the spectra into other software, for example to create a custom searchable library in the NIST MS Search program. The PDF reports and MSP files are available below (through links to Dropbox). MD5 checksums are given to verify the file integrity.

Within the R packages, the mass spectra (m/z values, intensities, and fragment ion identifications) and metatdata (compound names, retention times, compound classes, etc.) are stored in CSV format for working with the data in R or exporting it to other software. The packages may contain functions to generate summary tables and plots. The mass spectral library packages depend on OrgMassSpecR and png.

California Sea Lion Contaminant Library (Example)

This library is provided as proof-of-concept. It contains a vignette (PDF) describing the general file structure and functionality of spectral library packages.

PDF Report: SpecLibExample.pdf
MD5 checksum: aa07e28848b7bff311d6dcdeeb097fc1
NIST MSP Text Format: SpecLibExample.txt
MD5 checksum: 31da4996d34b7c6da9635608261b5890
R Package: SpecLibExample
Authors: Eunha Hoh and Nathan Dodder
Sample: Blubber from a male sea lion (Zalophus californianus) fatally stranded in the year 2000 in southern California.
Instrument: GCxGC-TOF in EI mode. A description of the analytical method is in the package help.
Compounds: 35 anthropogenic contaminants, 3 suspected natural halogenated products, and 1 unknown.

Massachusetts Dolphin Blubber Contaminant Library

The original PDF version of this library appears appears as supporting information for the manuscript referenced below. Note the PDF version here has different page numbers. Documentation is in the manuscript, supporting information, and the R package.

PDF Report: SpecLibDolphin2011.pdf
MD5 checksum: a8bf4a3616fe5db5bf0cb0b09d25f538
NIST MSP Text Format: SpecLibDolphin2011.txt
MD5 checksum: 853fdf8a8b1d5300d59dd9c0e653e519
R Package: SpecLibDolphin2011
Reference: "Nontargeted Comprehensive Two-Dimensional Gas Chromatography/Time-of-Flight Mass Spectrometry Method and Software for Inventorying Persistent and Bioaccumulative Contaminants in Marine Environments", Environmental Science and Technology, 2012, 46 (15), 8001-8008. DOI: 10.1021/es301139q.
Authors: Eunha Hoh, Nathan G. Dodder, Steven J. Lehotay, Kristin C. Pangallo, Christopher M. Reddy, and Keith A. Maruya
Sample: Blubber from a common dolphin (Delphinus delphis) fatally stranded January 2006 in Orleans, Massachusetts.
Instrument: GCxGC-TOF in EI mode. A full description of the analytical method is in the above literature reference.
Compounds: The data set contains a total of 271 identified compounds belonging to 24 structural classes. All compounds but one were halogenated. Eighty-six compounds were anthropogenic contaminants that are not routinely targeted in environmental surveys, and 54 compounds were halogenated natural products. Spectra for the 55 PCB congeners are not included in the library. The library also contains 27 halogenated mass spectra that were not able to be identified.
